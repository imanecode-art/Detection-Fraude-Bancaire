# Detection-Fraude-Bancaire
# Détection de Fraude Bancaire - Projet Académique
Ce projet utilise le Machine Learning (XGBoost, Random Forest) pour détecter les transactions frauduleuses.

## Caractéristiques
- *Modèle :* XGBoost avec Recall de 82%.
- *Outils :* Python, Scikit-Learn, Pandas.
- *Stabilité :* Vérifiée par Validation Croisée (K-Fold)
